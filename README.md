# 📘 Employee Management System

A Java-based web application for managing employee records using Servlets, JSP, JDBC, and MySQL, built with a clean layered architecture.

---

## 🚀 Features

- Add, View, Update, Delete Employees (CRUD Operations)
- User Authentication (Login & Registration)
- Layered Architecture (Controller, DAO, DTO, Utility)
- JDBC-based Database Connectivity
- Simple UI using HTML & CSS

---

## 🛠️ Tech Stack

- Backend: Java, JDBC, Servlets, JSP  
- Frontend: HTML, CSS  
- Database: MySQL  
- Server: Apache Tomcat  
- IDE: Eclipse
